6
Marketing Engine — Mailtrap Integration
📬

Transaktionsmails

Ticket-Bestätigung, Session-Buchung, Passwort-Reset — via Mailtrap Sending API. Anhänge (Ticket PDF) direkt mitgesendet.

📢

Marketing-Kampagnen

Kampagnen-Erstellung im Admin, Versand via Mailtrap Marketing API. Stats werden zurückgeholt und im Dashboard angezeigt.

👥

Kontakt-Import

CSV-Import der bestehenden Kontakte aus Event 2025. Automatische Synchronisation neuer Ticketkäufer. Tags für Segmentierung.

🔗

UTM-Tracking

Pro Kampagne UTM-Parameter + Shortlink generieren. Ticket-Käufe werden der auslösenden Kampagne zugeordnet.

Implementierungs-Prompt — Woche & Task anpassen, dann kopieren 
Du bist Senior Full-Stack Engineer bei go2flow.

Wir bauen die D2C Summit Platform — eine vollständige Event Management
Platform für den jährlichen D2C Summit.

TECH STACK (go2flow Standard, keine Abweichungen):
- Backend: Laravel (LTS) + MySQL
- Frontend: Nuxt 4 / Vue 3
- Styling: Tailwind CSS (ausschließlich, kein SCSS/CSS)
- Auth: Laravel Sanctum
- Email: Mailtrap API (transaktional + marketing)
- Payment: Payrexx Whitelabel
- Storage: Laravel Storage (S3-kompatibel)
- Tests: Pest (Laravel), Vitest (Nuxt)

ARCHITEKTUR:
Single Source of Truth im Laravel Backend. Alle Frontends
(Public Website, My Summit, Admin, Sponsor-Portal, Speaker-Portal,
Check-in) ziehen aus derselben MySQL-Datenbank via REST API.

DOMAIN-STRUKTUR:
- d2c-summit.ch        → Public Website
- my.d2c-summit.ch     → My Summit + alle Portale
- admin.d2c-summit.ch  → Admin Backend
- checkin.d2c-summit.ch → Check-in App
