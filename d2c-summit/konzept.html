01

Vision

Projektziel
Eine vollständige, wiederverwendbare Event Management Platform für den D2C Summit — jährlich durchführbar, ohne jedes Mal von vorne zu beginnen. Ein einziges Laravel-Backend als Single Source of Truth für alle Frontends. Was einmal konfiguriert ist (Programm, Speaker, Tickets, Sponsoren), erscheint automatisch überall — auf der Public Website, im My Summit, in der Check-in App.
Das erste Event (2025) wurde bereits durchgeführt. Die neue Platform wird jetzt Greenfield gebaut — mit dem Ziel, jedes Jahr dasselbe System zu nutzen und nur die Inhalte zu aktualisieren. Kein Wegwerfprojekt, sondern ein dauerhaftes Werkzeug.

Rollen & Zugang

Rolle Zugang Portal
super_admin Alles — Events anlegen, Rollen vergeben, Finanz-Admin Admin Backend
organizer Event-Inhalte, Programm, Speaker, Tickets (kein Finanz-Admin) Admin Backend
sponsor_manager Sponsor-Pakete definieren, Sponsoren zuweisen Admin Backend
attendee My Summit: Ticket, Sessions buchen, Badge, Materialien My Summit
speaker Eigenes Profil bearbeiten, Präsentationen hochladen, Feedback Speaker-Portal
sponsor Firmenprofil, Logo-Upload, Materialien-Download, Stats Sponsor-Portal
checkin Nur QR-Scanner und Einlass-Statistiken — kein sonstiger Zugang Check-in App
Wichtig: Rollen sind per Event vergeben. Ein Nutzer kann in Event 2025 Organizer sein und in Event 2026 ebenfalls — oder verschiedene Rollen in verschiedenen Events haben.
05

Tech Stack

Schicht Technologie Begründung
Backend API Laravel (LTS) + MySQL go2flow Standard — robust für komplexe Datenmodelle
Frontend Nuxt 4 / Vue 3 go2flow Standard — SSR für Public Website, SPA für Portale
Styling Tailwind CSS (latest) go2flow Pflicht — kein SCSS / kein Custom CSS
State Management Pinia Vue 3 Standard
Auth Laravel Sanctum Token-basiert, einfach mit Nuxt integrierbar
E-Mail Mailtrap API Transaktional + Marketing — bestehender Account vorhanden
Zahlung Payrexx Whitelabel Bestehende go2flow Integration
File Storage Laravel Storage + S3 Logos, PDFs, Präsentationen
PDF-Generierung Browsershot / DomPDF Tickets, Badges, Zertifikate
Tests Pest (Laravel) + Vitest (Nuxt) go2flow Standard
CI/CD GitHub Actions go2flow Standard
