D2C Summit / Module Phase I
Module Phase I
Alle 14 Module mit vollständigen Feature-Beschreibungen — was die Platform können muss
A

Core Platform — Event, Programm & Speaker

🔁
Modul 01
Event Engine
Jährlich wiederkehrende Events verwalten — Single Source of Truth für alle Inhalte
  • Event erstellen: Name, Slug, Datum, Venue, Beschreibung, Status (draft/published/archived)
  • Branding-Config pro Event: Primärfarbe, Logo, Hero-Image
  • Stages / Räume definieren (Name, Kapazität, Reihenfolge)
  • Event klonen — für Folgejahre als Ausgangspunkt
  • Vergangene Events archivieren, Statistiken bleiben erhalten
🗓️
Modul 02
Programm-Manager
Einmal erfassen → überall anzeigen (Website, My Summit, Admin)
  • Session-Typen: Keynote / Masterclass / Panel / Break / Networking
  • Session anlegen: Titel, Beschreibung, Typ, Stage, Start/End, Kapazität, buchbar
  • Speaker einer Session zuweisen (mehrere möglich)
  • Session-Materialien hochladen (Präsentation, Ressourcen)
  • Drag & Drop Programm-Reihenfolge im Admin
  • Parallele Sessions sichtbar im Tagesplan
🎤
Modul 03
Speaker-Management
Speaker-Verwaltung im Admin + Self-Service über Speaker-Portal
  • Speaker-Profil: Name, Titel, Firma, Bio, Foto, LinkedIn, Twitter, Website
  • Speaker einem Event zuweisen, Sessions verknüpfen
  • Speaker-Login: Eigenes Profil bearbeiten, Präsentationen hochladen
  • Eigenes Session-Feedback einsehen (nach dem Event)
  • Automatische Anzeige auf Public Website + My Summit
B

Ticketing & Mitgliederbereich

💳
Modul 04
Ticket-System via Payrexx
Vollständiger Ticket-Kauf via Payrexx Whitelabel-Checkout
  • Ticket-Kategorien pro Event: Name, Preis, Kapazität, Verkaufszeitraum
  • Persönliches Ticket: Vorname, Nachname, Firma, Funktion eingedruckt
  • Checkout: Kategorie → Daten → Payrexx → Bestätigung
  • QR-Code pro Ticket (unique, fälschungssicher)
  • Ticket-PDF generieren + per E-Mail senden
  • Order-Management: Bestellübersicht, Status, Rechnungsdownload
  • Stornierung + Rückerstattung (manuell via Admin)
🎫
Modul 04b
Voucher-System
Rabatt-Codes und Freikarten — mit Direktlink-Funktion
  • Voucher-Typen: Prozent-Rabatt / Fixbetrag / 100% Gratis
  • Max. Einlösungen pro Code + Ablaufdatum definierbar
  • Beschränkung auf bestimmte Ticket-Kategorie (optional)
  • Direktlink: tickets?voucher=CODE → vorausgefüllter Checkout
  • Voucher-Statistiken: wer hat wann eingelöst
🏠
Modul 05
My Summit (Mitgliederbereich)
Persönlicher Bereich nach Ticketkauf — das Zuhause des Attendees
  • Dashboard: Ticket, Tagesplan, offene Buchungen, neue Materialien
  • Ticket-Ansicht: QR-Code, Daten, Kategorie, Rechnung-Download
  • Programm & Buchung: Masterclasses mit Platz-Anzeige buchen/stornieren
  • Warteliste: bei voller Session + automatische Benachrichtigung
  • Materialien: Präsentationen der gebuchten/besuchten Sessions
  • Profil bearbeiten: Name, Firma, Funktion, LinkedIn
C

Rollen & Portale

🔐
Modul 06
Rollen & Rechte
7 Rollen, per Event vergeben — granulare Zugangssteuerung
  • super_admin / organizer / sponsor_manager / attendee
  • speaker / sponsor / checkin
  • Rollen sind pro Event vergeben (nicht global)
  • Middleware-basierter API-Schutz in Laravel
  • Rollen-Verwaltung durch super_admin im Backend
🏢
Modul 07
Sponsor-Portal
Self-Service für Sponsoren + Verwaltung durch Admin
  • Sponsor-Pakete: Platin / Gold / Silver / Bronze / Partner
  • Leistungen pro Paket (JSON): Logo-Platz, Tickets, Redner-Slot, etc.
  • Self-Service: Logo hochladen (PNG/SVG), Firmenprofil, Website-URL
  • Materialien-Download: Toolkit, Design-Guidelines (vom Admin hinterlegt)
  • Statistiken: Logo-Impressions auf Website, Link-Klicks
  • Public Website: automatische Anzeige nach Paket-Rang (Platin oben)
D

Marketing & Statistiken

📧
Modul 08
Marketing Engine
Vollständiges E-Mail Marketing + automatisierte Sequenzen via Mailtrap
  • Kontakt-Management: CSV-Import, Auto-Import aus Ticketkäufen, Tags, DSGVO
  • Kontaktlisten / Segments erstellen (z.B. "VIP", "2025-Attendee")
  • Kampagnen: Name, Betreff, Template, Ziel-Liste, geplant/sofort
  • HTML-Template-Editor + Variablen ({{first_name}} etc.)
  • Automatisierte Sequenzen: Ticket gekauft → Session gebucht → Pre-Event → Post-Event
  • UTM-Tracking: Pro Kampagne, Shortlinks, Klick → Konversion
  • Versand-Statistiken: Sent / Opened / Clicked / Bounced
📊
Modul 09
Statistics Dashboard
Echtzeit-Übersicht für Organizer — vor, während und nach dem Event
  • Ticket-Verkäufe: gesamt + nach Kategorie (Zahl + Chart)
  • Umsatz gesamt, Voucher-Einlösungen
  • Registrierungen vs. Check-ins am Eventtag
  • Session-Auslastung (welche Masterclasses voll/leer)
  • Marketing-Stats: Kampagnen-Performance, UTM-Conversions
  • Post-Event: Check-in Rate, Session-Attendance, Feedback-Ratings
  • Vorjahresvergleich (wenn Event 2025 archiviert vorhanden)
E

Tools & Generierte Inhalte

📱
Modul 10
Check-in Web-App
Mobiloptimierte QR-Scanner-App für das Einlass-Team am Eventtag
  • Zugang nur mit Rolle checkin — kein weiterer Zugang
  • QR-Code Scanner via Browser-Kamera (kein App-Download nötig)
  • Ticket validieren: Name, Firma, Kategorie, gebuchte Masterclasses
  • Check-in bestätigen → Status wird sofort auf checked_in gesetzt
  • Fehlermeldung: ungültiger QR / bereits eingecheckt / storniert
  • Offline-Fallback: lokaler Cache für Notfälle
  • Live-Stats: Einlass-Fortschritt für das Team
🌐
Modul 11
Public Website
SSR-Website komplett aus dem Backend gespeist — kein CMS nötig
  • / — Hero, Event-Info, Highlights, Early-Bird-Teaser, Sponsor-Logos
  • /programm — Tagesplan (Timeline + Filter nach Stage/Typ)
  • /speaker — Speaker-Grid mit Detailseite
  • /sponsoren — Logos nach Paket-Rang mit Link
  • /tickets — Kategorien + Payrexx-Checkout + Voucher-Feld
  • /faq — pflegbar im Admin
  • SEO: SSR via Nuxt 4, Schema.org Event-Markup, OG-Tags
🃏
Modul 12
Social Share Cards
Auto-generierte Sharing-Bilder nach Ticketkauf
  • LinkedIn/OG-Format: 1200×630px
  • Instagram-Format: 1080×1080px
  • Inhalt: "Ich bin dabei! · D2C Summit [Jahr] · [Datum] · [Ort]"
  • Branding: Event-Logo, Primärfarbe, Name des Attendees
  • Downloadbar in My Summit
  • Beim Ticketkauf direkt angeboten ("Share on LinkedIn")
🏷️
Modul 13
Attendee Badge
Personalisiertes Event-Badge — für LinkedIn und Druck
  • Design pro Ticket-Kategorie (VIP-Badge sieht anders aus)
  • Inhalt: Name, Firma, Funktion, Kategorie, QR-Code
  • PDF (A6, druckfertig) + PNG (LinkedIn-optimiert)
  • Badge-Vorlagen im Admin konfigurierbar (template_config JSON)
  • Badge auch vor Ort ausdruckbar (Admin-Funktion)
📜
Modul 14
Zertifikat & Feedback
Certificate of Attendance + Session-Bewertungen nach dem Event
  • Certificate of Attendance: PDF, nach Event in My Summit verfügbar
  • Session-Feedback via E-Mail-Trigger (Tag +1 nach Event)
  • Bewertung: 1–5 Sterne + optionaler Kommentar
  • Speaker sieht eigenes Feedback in Speaker-Portal
  • Admin sieht alles aggregiert in Statistics Dashboard
