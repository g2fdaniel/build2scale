A

Core Platform — Event, Programm & Speaker

🔁
Modul 01MVP
Event Engine
Events anlegen und verwalten — Single Source of Truth für alle Inhalte
  • Event erstellen: Name, Datum, Venue, Beschreibung, Status (draft/published/archived)
  • Branding-Config: Logo, Primärfarbe, Hero-Image
  • Stages / Räume definieren (Name, Kapazität, Reihenfolge)
  • Vergangene Events archivieren, Daten bleiben erhalten
  • Event klonen für Folgejahre
🗓️
Modul 02MVP
Programm-Manager
Einmal erfassen → überall anzeigen (Website, My Summit, Admin)
  • Session-Typen: Keynote / Masterclass / Panel / Break / Networking
  • Session anlegen: Titel, Beschreibung, Typ, Stage, Start/End, Kapazität, buchbar
  • Speaker einer Session zuweisen (mehrere möglich)
  • Programm-Reihenfolge manuell festlegen
  • Parallele Sessions sichtbar im Tagesplan
  • Session-Materialien hochladen (Präsentationen)
  • Drag & Drop Reihenfolge im Admin
🎤
Modul 03MVP
Speaker-Management
Speaker-Verwaltung im Admin — Self-Service Portal kommt Post-MVP
  • Speaker-Profil: Name, Titel, Firma, Bio, Foto, LinkedIn, Website
  • Speaker einem Event zuweisen, Sessions verknüpfen
  • Automatische Anzeige auf Public Website
  • Speaker-Login: Profil selbst bearbeiten
  • Präsentationen selbst hochladen
  • Session-Feedback einsehen
B

Unternehmen & Teilnehmer Neu

🏢
Modul 04MVP
Unternehmens-Profile
Unternehmen als eigene Entität — Tickets zuordnen, Logo, Website-Sichtbarkeit steuern
  • Unternehmen anlegen: Name, Logo (PNG/SVG), Website, Beschreibung, Branche
  • Typ: Sponsor (mit Paket: Platin/Gold/Silver/Bronze) oder Attendee-Unternehmen
  • Sichtbarkeit auf Website: Sponsor-Seite / Partner-Bereich / unsichtbar
  • Mehrere Tickets einem Unternehmen zuordnen (z.B. 2 Tickets → Brand X)
  • Teilnehmer-Liste pro Unternehmen einsehbar im Admin
  • Öffentliches Profil: Firmenlogo nach Rang sortiert auf Sponsoren-Seite
  • Admin-Übersicht: Filter nach Typ, Event, Sichtbarkeit, Paket
👥
Modul 05MVP
Teilnehmer-Verwaltung
Alle Attendees im Überblick — mit Unternehmens-Verknüpfung und Kommunikations-Status
  • Teilnehmer-Übersicht: Name, Firma, Ticket-Kategorie, Check-in-Status
  • Unternehmens-Zuordnung: Teilnehmer ↔ Unternehmen verknüpfen
  • Filter: nach Unternehmen, Ticket-Kategorie, Check-in-Status
  • Kommunikations-Status: Wer hat welche Mail erhalten / geöffnet
  • Teilnehmer-Export (CSV) für externe Tools
  • Manuelles Hinzufügen / Bearbeiten von Teilnehmer-Daten
💳
Modul 06MVP
Ticket-System via Payrexx
Vollständiger Ticket-Kauf via Payrexx — mit QR-Code und Bestätigungsmail
  • Ticket-Kategorien: Name, Preis, Kapazität, Verkaufszeitraum
  • Checkout: Kategorie → Persönliche Daten → Payrexx → Bestätigung
  • QR-Code pro Ticket (unique, fälschungssicher)
  • Ticket-PDF + Bestätigungsmail mit Anhang
  • Order-Management: Status, Rechnungsdownload
  • Stornierung + Rückerstattungs-Workflow (manuell via Admin)
  • Voucher-System: %-Rabatt, Gratis-Tickets, Direktlink (?voucher=CODE)
C

Rollen, Admin & Portale

🔐
Modul 07MVP
Rollen & Rechte
Vollständige Berechtigungs-Architektur von Tag 1 — pro Event vergeben
  • 7 Rollen: super_admin / organizer / sponsor_manager / attendee / speaker / sponsor / checkin
  • Rollen sind pro Event vergeben (nicht global — gleiche Person, verschiedene Rollen)
  • Rollenbasierter Zugriffsschutz auf allen 4 Domains
  • Rollen-Vergabe durch super_admin im Backend
  • Erweiterbar: neue Rollen ohne Architektur-Änderung
🪪
Modul 08MVP — Basis
My Summit
Fundament des Teilnehmer-Portals — vollständiger Ausbau Post-MVP
  • Login / Registrierung (verknüpft mit Ticketkauf)
  • Ticket anzeigen: QR-Code, Daten, Kategorie, Rechnung-Download
  • Persönliches Profil bearbeiten (Name, Firma, Funktion)
  • Programm-Übersicht (nur Lese-Ansicht im MVP)
  • Session-Buchung & Warteliste
  • Materialien & Präsentationen
  • Badge-Download, Social Share Cards, Zertifikat
⭐
Modul 09MVP
Sponsor-Verwaltung
Sponsoren anlegen und auf Website steuern — über Unternehmens-Profil (Modul 04)
  • Sponsor-Pakete definieren: Platin / Gold / Silver / Bronze / Partner
  • Unternehmen als Sponsor markieren + Paket zuweisen
  • Logo-Upload direkt im Admin (kein Sponsor-Login nötig)
  • Website-Anzeige: automatisch nach Paket-Rang sortiert
  • Sichtbarkeit pro Sponsor steuerbar (öffentlich / versteckt)
  • Sponsor Self-Service Portal (eigenes Login)
  • Logo-Impressions & Klick-Statistiken
D

Kommunikation & Statistiken

📧
Modul 10MVP
Kommunikations-Basis
Transaktionale Mails + gezielte Teilnehmer-Updates — via Mailtrap
  • Transaktionale Mails: Ticket-Bestätigung (mit PDF), Passwort-Reset, Systembenachrichtigungen
  • Gezielte Updates: Admin kann Mails an Teilnehmer-Gruppen senden (z.B. "Masterclasses jetzt buchbar")
  • Empfänger-Filter: alle Attendees / nach Ticket-Kategorie / nach Unternehmen
  • E-Mail Templates im Admin pflegbar (Variablen: Name, Ticket, etc.)
  • Versand-Übersicht: wer hat was erhalten
  • Automatisierte Sequenzen (Trigger-basiert)
  • Kampagnen-Manager + UTM-Tracking
📊
Modul 11MVP — Basis
Statistiken
Echtzeit-Überblick für den Organizer — Ticket-Sales und Check-in
  • Ticket-Verkäufe: Anzahl gesamt + nach Kategorie
  • Umsatz gesamt + Voucher-Einlösungen
  • Check-in Rate am Eventtag (live)
  • Teilnehmer-Liste mit Unternehmens-Zuordnung
  • Session-Auslastung & Feedback-Ratings
  • Marketing-Stats & UTM-Conversions
  • Vorjahresvergleich
E

Frontends & Event-Tools

🌐
Modul 12MVP
Public Website
Vollständige öffentliche Eventseite — aus dem Backend gespeist, kein CMS nötig
  • / — Hero, Event-Info, Highlights, Sponsor-Logos
  • /programm — Tagesplan (Timeline + Filter nach Stage/Typ)
  • /speaker — Speaker-Grid mit Detailseiten
  • /sponsoren — Unternehmen nach Paket-Rang + Logos
  • /tickets — Kategorien + Payrexx-Checkout + Voucher-Feld
  • /faq — pflegbar im Admin
  • SEO: SSR, Schema.org Event-Markup, OG-Tags
📱
Modul 13MVP
Check-in Web-App
Mobiloptimierter QR-Scanner für das Einlass-Team — kein App-Download nötig
  • Zugang nur mit Rolle checkin
  • QR-Code Scanner via Browser-Kamera
  • Ticket validieren: Name, Firma, Kategorie
  • Fehlermeldung: ungültiger QR / bereits eingecheckt / storniert
  • Live-Einlass-Zähler für das Team
  • Offline-Fallback (lokaler Cache)

II

Post-MVP — Phase II

🪪
Modul 08+Post-MVP
My Summit — Vollausbau
Alles was nach dem ersten Event gebraucht wird
  • Masterclass-Buchung mit Platz-Anzeige + Warteliste
  • Automatische Benachrichtigung bei freiem Platz
  • Materialien & Präsentationen nach dem Event
  • Badge-Download (PDF + PNG)
  • Social Share Cards (LinkedIn + Instagram)
  • Certificate of Attendance (PDF)
🎤
Modul 03+Post-MVP
Speaker Self-Service Portal
Speaker können ihr Profil und Materialien selbst pflegen
  • Speaker-Login (eigenes Portal unter my.d2c-summit.ch)
  • Profil selbst bearbeiten (Bio, Foto, Social Links)
  • Präsentationen hochladen
  • Session-Feedback einsehen (nach Event)
⭐
Modul 09+Post-MVP
Sponsor Self-Service Portal
Sponsoren pflegen ihr Profil selbst — inkl. Statistiken
  • Sponsor-Login (eigenes Portal unter my.d2c-summit.ch)
  • Logo selbst hochladen + Firmenprofil bearbeiten
  • Sponsor-Materialien downloaden (vom Admin hinterlegt)
  • Logo-Impressions & Klick-Statistiken einsehen
📧
Modul 10+Post-MVP
Marketing Engine
Vollständige Kampagnen + automatisierte Sequenzen via Mailtrap
  • Kontakt-Segmentierung (Tags, Listen, Attribute)
  • Kampagnen: planen, senden, Statistiken
  • Automatisierte Sequenzen (Ticket gekauft → Pre-Event → Post-Event)
  • UTM-Tracking + Klick → Konversion
  • Versand-Statistiken: Sent / Opened / Clicked / Bounced
📊
Modul 11+Post-MVP
Statistics Dashboard — Voll
Vollständige Analytics — Marketing, Sessions, Vorjahresvergleich
  • Session-Auslastung (welche Masterclasses voll/leer)
  • Session-Feedback aggregiert
  • Marketing-Stats: Kampagnen-Performance, UTM-Conversions
  • Vorjahresvergleich (ab Event 2)
🔁
Modul 01+Post-MVP
Event-Klon & Multi-Year
Für Folgejahre — Event klonen, Inhalte übernehmen, neu starten
  • Event klonen als Ausgangspunkt für nächstes Jahr
  • Vergangene Events archivieren (Statistiken bleiben)
  • Branding + Programm aus Vorjahr als Template
