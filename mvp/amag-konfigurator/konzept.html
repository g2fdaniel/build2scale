01

Scope — Was ist im MVP enthalten?

Im Scope enthalten
  • Option A: Standalone LP (Subdomain konfigurator.amag.ch)
  • Option B: Embed-Widget für amag.ch/zubehoer
  • Fahrzeug-Konfigurator (Marke / Modell / Jahr)
  • Produktzuordnung via CSV-Import
  • 5–8 Zubehör-Kategorien (AMAG-Auswahl)
  • Strukturiertes Anfrage-Formular
  • Mail-Benachrichtigung bei Anfrage
  • Mobile-optimiert
  • Analytics-Integration (Google Analytics / GA4)
  • Admin-Panel: Produkte & Zuordnungen
Nicht im Scope (MVP)
  • Shop / Warenkorb / Payment
  • AMAG-Backend-Anbindung (SAP, CRM)
  • Lagerbestand-Echtzeit-Sync
  • Kundenkonto / Login
  • Mehrsprachigkeit Phase 1 (optional)
  • Garantie- / Einbauverwaltung
  • Integration amag.ch CMS
Deployment-Optionen: Option A (Subdomain) — eigenständig, kein IT-Aufwand AMAG  ·  Option B (Embed) — 3 Zeilen Code auf amag.ch, läuft auf go2flow-Infrastruktur, AMAG-IT nur CSP-Freigabe. Beide Optionen im Fixpreis enthalten.
